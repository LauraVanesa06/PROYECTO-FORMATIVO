Sobre el Proyecto

S.I.E - Sistema Integral de Empresas

S.I.E. es una solución integral diseñada para la gestión eficiente de inventarios y ventas en empresas ferreteras. Está conformado por una página web y una aplicación móvil que trabajan de forma complementaria.

La página web, desarrollada en Ruby on Rails, cuenta con dos módulos principales: uno administrativo, que permite gestionar productos, existencias y procesos internos, y otro para clientes, donde se pueden explorar los productos y realizar compras en línea.

La aplicación móvil, desarrollada en Flutter, está orientada al cliente final y ofrece una interfaz intuitiva y moderna para visualizar productos, consultar detalles y efectuar pedidos.

En conjunto, S.I.E. busca optimizar la administración del inventario, mejorar la eficiencia operativa y brindar una experiencia de compra ágil y práctica para los usuarios.

Alcance y Restricciones del Proyecto

Definimos claramente los límites y capacidades de nuestro sistema

Alcance del Proyecto

  • Gestión completa de productos: registro, actualización, eliminación y control de existencias.
  • Visualización del inventario en tiempo real para administradores.
  • Catálogo digital interactivo con detalles y precios actualizados.
  • Sistema de compras en línea sencillo y seguro (web y móvil).
  • Reportes y estadísticas básicas sobre inventario y ventas.
  • Interfaz responsiva adaptada a diferentes dispositivos.

Restricciones del Proyecto

  • Depende de conexión a internet estable para sincronización de datos.
  • No contempla integración inicial con sistemas contables externos.
  • App móvil limitada a funciones cliente (sin administración interna).
  • Pagos en línea sujetos a servicios de terceros disponibles.
  • Infraestructura del servidor proporcionada por la empresa usuaria.
